   .. method:: object_to_numeric(self, data, scale=None, offset=None)

      Converts the object type elements of a pandas series to numeric type.
      Right spaces are trated as ceros. Scale and offset can optionally be applied.
      The final data type according to the class dtype.

      :param self: Pandas dataframe with a column per report sections.
                   The sections in the columns as a block strings.
      :type self: dtype, numeric_scale and numeric_offset
      :param data: Series with data to convert. Data must be object type
      :type data: pandas.Series

      :keyword scale: Scale to apply after conversion to numeric
      :kwtype scale: numeric, optional
      :keyword offset: Offset to apply after converion to numeric
      :kwtype offset: numeric, optional

      :returns: **data** -- Data series of type self.dtype
      :rtype: pandas.Series
